# 首页

### 社区简介

开服专区是《Minecraft》小型服务器游戏。目前已更替了许多周目。服务器最早于2016年就已成立，服务器最初为早期人类间进行Minecraft PE版联机，而后在2017年左右不断进化为基岩版、2018年进化为《Minecraft》JAVA版（PC平台）， 且一直为玩家提供服务器级特色玩法。
